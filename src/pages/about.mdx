## 关于我

我是谁？

在经过几次推到后，我发现我很难回答这个问题了。我曾经以为我是别人眼中的定义，别人认为我是谁，我“才”是谁。这明显不对了，因为我把我自己的存在抛弃了。尴尬的是，推翻这个错误想法后，我又难以定义自己了——我到底是谁呢？我不喜欢给人分类的“人格测试”，也不喜欢那些给人贴标签的惯性思维。也许我内心对我有一个定义，但是我又不能自己去说……

所以，我只想告诉你，我只是在数据海洋中一个活生生的人。在这里我没有面具，这就是真实的我。

至于我到底是谁，看几篇博客，再给出你自己的定义吧。

## 关于博客

什么时候开始接触博客的？初一到初二的那个暑假。

这算是一个比较漫长的过程了，只是觉得能在网络上获得属于自己的一片小天地，就促使着我折腾：刚开始什么都不懂，买了个服务器，用米拓建站搞了一个站点……很简陋，甚至连域名都没有。

不过那段时间还是挺快乐的，用**EasyX**写的程序终于有一个地方可以放了，通过 IP 就可以访问，这对当时的我简直是大开眼界。

后面知道了**WordPress**，可惜我的服务器太差了，访问速度感人……我自己都受不了了。到最后服务器到期了，我的站点、我的程序也都被抹去了。

随后就是不要钱的**Hexo**，第一次知道静态站点又给我吓到了：这么方便！还有这么大的自由度，太酷了吧。

结果就是陷入了美化的怪圈。因为不会技术又不敢问，我就自己一次又一次地试——说来也奇怪，那个时候的我心里只有把这个效果移植过来的想法，却从来没有想过自己学点技术、自己改改。从我 2023 年 Github 的提交次数也看得出来，这种事情占去了我大量的时间，我也成为了别人口中“只美化不写博客”的人。

最后遇到了**Astro**，也就是现在我使用的**Web 框架**。其实刚开始我是有点犹豫的，毕竟**Hexo**已经让我浪费了 1000+次 commit 了，这个沉没成本还是有点大了。不过结果都已经知道了，我还是换了**Astro**， 动机已经不是为了“在网络上获得属于自己的一片小天地”，具体是什么，我也很难讲得清楚。

现在，我可以自豪地说，我终于不再是之前哪个什么都不会的小屁孩，我给自己写了一个主题，这就是最好的证明！
